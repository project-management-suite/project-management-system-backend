# project-management-backend

Bootstrap project directory created by bootstrap.sh.

Next steps:
1. Run `npm init` in this folder to create package.json (you requested starting with npm init).
2. Install dependencies (example):
   npm install express mongoose dotenv bcryptjs jsonwebtoken cors helmet morgan express-validator multer nodemailer swagger-jsdoc swagger-ui-express dayjs node-cron uuid
3. Copy .env.example -> .env and fill secrets.
4. Implement controllers, routes, and models in src/.
5. Start the server:
   node server.js
   or (dev) nodemon server.js
